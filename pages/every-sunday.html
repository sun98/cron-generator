Cron Every Sunday

Expression: 0 0 * * 0

What Does 0 0 * * 0 Mean?

  • 0 = Minute 0
  • 0 = Hour 0 (midnight)
  • * = Every day of month
  • * = Every month
  • 0 = Sunday (0 = Sunday in cron)

Common Use Cases

💾 Weekly Backups

  • 0 0 * * 0 /scripts/weekly-backup.sh - Full system backup on Sunday
  • 0 1 * * 0 /scripts/backup-db-weekly.sh - Database backup at 1 AM Sunday

📊 Weekly Reports

  • 0 0 * * 0 /scripts/weekly-report.sh - Generate weekly analytics
  • 0 8 * * 0 /scripts/send-weekly-summary.sh - Send weekly summary at 8 AM

🧹 Maintenance

  • 0 3 * * 0 /scripts/deep-cleanup.sh - Deep cleanup on Sunday morning
  • 0 4 * * 0 /scripts/rebuild-index.sh - Rebuild search indexes weekly
