Crontab Syntax

Basic crontab line format:

* * * * * command

Examples

  • */15 * * * * /script.sh - Run every 15 min
  • 0 5 * * * /backup.sh - Daily 5 AM backup
  • 0 0 * * 0 /weekly.sh - Weekly Sunday
