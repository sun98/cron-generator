Cron Expression Format

A cron expression has 5 fields: minute hour day month weekday

Field Ranges

  • Minute: 0-59
  • Hour: 0-23
  • Day: 1-31
  • Month: 1-12
  • Weekday: 0-6 (0=Sunday)

Special Characters

  • * - Any value
  • , - Value list separator (1,3,5)
  • - - Range of values (1-5)
  • / - Step values (*/5)
